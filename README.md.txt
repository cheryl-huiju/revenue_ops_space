# Custom Pricing Tool Project

## Overview
This project aims to launch a **Custom Pricing Tool** designed to help Account Managers (AMs) increase cross-selling opportunities and reduce seller churn. The tool provides transparency in custom rate applications, improves coaching data visibility for AM Leadership, and streamlines workflows for operational teams.

## Objective
To enable AMs to apply custom processing rates effectively, increasing monthly revenue per seller by 10% and reducing churn to 0.3% monthly. Additionally, the tool ensures seamless bug reporting and adoption rates within parameters set by AM Leads.
